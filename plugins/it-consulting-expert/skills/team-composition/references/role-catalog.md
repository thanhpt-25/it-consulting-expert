## Management Roles

### Project Manager (PM / プロジェクトマネージャー)
- **Seniority**: Senior (10+ years experience, 5+ years PM)
- **Responsibilities**: Overall project delivery, scope/schedule/budget management, client relationship, risk management, team leadership, escalation handling
- **Skills**: PMP/PRINCE2, communication, negotiation, financial management, Japanese business etiquette
- **Allocation**: 100% throughout project
- **Typical ratio**: 1 PM per 10-15 team members

### Assistant Project Manager (副PM)
- **Seniority**: Mid-Senior (7+ years)
- **Responsibilities**: Support PM, track detailed progress, coordinate across teams
- **When needed**: Projects with 15+ team members or multiple workstreams

### PMO (Project Management Office)
- **Seniority**: Mid (5+ years)
- **Responsibilities**: Standards, templates, reporting, resource tracking
- **When needed**: Programs with multiple sub-projects

## Engineering Roles

### Technical Lead (TL / テクニカルリード)
- **Seniority**: Senior (8+ years, deep expertise in primary stack)
- **Responsibilities**: Architecture decisions, technical mentoring, code quality standards, technology selection, technical risk assessment
- **Skills**: System design, code review, performance optimization, security awareness
- **Allocation**: 80-100% (may split between hands-on and leadership)
- **Typical ratio**: 1 TL per 5-8 developers

### Business Analyst (BA / ビジネスアナリスト)
- **Seniority**: Mid-Senior (5+ years)
- **Responsibilities**: Requirements elicitation, business process modeling, use case writing, stakeholder management, acceptance criteria definition
- **Skills**: Business domain knowledge, process modeling (BPMN), documentation, facilitation
- **Allocation**: 100% during requirements, 50% during design, 20-30% during development
- **When needed**: All projects with complex business logic

### System Engineer - Senior (SE上級 / 上級システムエンジニア)
- **Seniority**: Senior (7+ years)
- **Responsibilities**: Detailed system design, complex module implementation, design review, junior mentoring
- **Skills**: Full-stack capability, design patterns, database design, API design
- **Allocation**: 100% during design and implementation phases

### System Engineer (SE / システムエンジニア)
- **Seniority**: Mid (3-7 years)
- **Responsibilities**: Module design, implementation, unit testing, documentation
- **Skills**: Primary language/framework proficiency, SQL, testing
- **Allocation**: 100% during implementation, 50-80% during design

### Programmer (PG / プログラマー)
- **Seniority**: Junior-Mid (0-5 years)
- **Responsibilities**: Coding to specifications, unit testing, bug fixing
- **Skills**: Programming fundamentals, version control, basic testing
- **Allocation**: 100% during implementation phase

### QA Engineer (テストエンジニア)
- **Seniority**: Mid (3+ years)
- **Responsibilities**: Test planning, test case design, test execution, defect reporting, test automation
- **Skills**: Test methodology, automation tools (Selenium, Cypress, JMeter), SQL for data verification
- **Allocation**: 30% during design (test planning), 100% during testing phases
- **Typical ratio**: 1 QA per 3-4 developers

### QA Lead (テストリード)
- **Seniority**: Senior (7+ years)
- **Responsibilities**: Test strategy, quality metrics, team coordination, client UAT support
- **When needed**: Projects with dedicated QA team of 3+

## Specialist Roles

### Infrastructure Engineer (インフラエンジニア)
- **Seniority**: Mid-Senior (5+ years)
- **Responsibilities**: Cloud architecture, server provisioning, networking, CI/CD, monitoring setup
- **Skills**: AWS/Azure/GCP, Docker, Kubernetes, Terraform, Ansible, networking
- **Allocation**: 50-100% depending on infra complexity

### DevOps Engineer (DevOpsエンジニア)
- **Seniority**: Mid-Senior (5+ years)
- **Responsibilities**: CI/CD pipelines, deployment automation, infrastructure as code, monitoring
- **Skills**: Jenkins/GitLab CI/GitHub Actions, Docker, K8s, scripting
- **When needed**: Projects requiring sophisticated deployment pipelines

### Security Engineer (セキュリティエンジニア)
- **Seniority**: Senior (7+ years)
- **Responsibilities**: Security architecture review, vulnerability assessment, compliance verification, security testing
- **Skills**: OWASP, penetration testing, SAST/DAST tools, compliance frameworks
- **Allocation**: 20-50% throughout, 100% during security review phases
- **When needed**: Finance, healthcare, government, or any project with sensitive data

### Database Administrator (DBA / データベース管理者)
- **Seniority**: Mid-Senior (5+ years)
- **Responsibilities**: Database design optimization, migration planning, performance tuning, backup strategy
- **Skills**: RDBMS (PostgreSQL, MySQL, Oracle, SQL Server), NoSQL, query optimization
- **When needed**: Data-intensive projects or complex migrations

### UX/UI Designer (UX/UIデザイナー)
- **Seniority**: Mid (3+ years)
- **Responsibilities**: User research, wireframing, visual design, prototyping, design system
- **Skills**: Figma/Sketch, user research methods, accessibility (WCAG), responsive design
- **Allocation**: 100% during design, 30-50% during implementation (developer support)
- **When needed**: Customer-facing applications

### Data Engineer (データエンジニア)
- **Seniority**: Mid-Senior (5+ years)
- **Responsibilities**: ETL pipelines, data warehouse design, data lake architecture
- **Skills**: SQL, Python, Spark, Airflow, cloud data services
- **When needed**: Analytics, BI, or data platform projects

### AI/ML Engineer (AI/MLエンジニア)
- **Seniority**: Mid-Senior (5+ years, ML specialization)
- **Responsibilities**: Model development, training pipeline, MLOps, AI integration
- **Skills**: Python, TensorFlow/PyTorch, MLflow, data preprocessing
- **When needed**: Projects with AI/ML components

### Change Management Consultant (チェンジマネジメントコンサルタント)
- **Seniority**: Senior (10+ years)
- **Responsibilities**: Organizational change planning, stakeholder alignment, training program design, adoption measurement
- **When needed**: Digital transformation projects affecting workflows

## Support Roles

### Technical Writer (テクニカルライター)
- **Seniority**: Mid (3+ years)
- **Responsibilities**: User manuals, API documentation, operation guides
- **Allocation**: 30-50% during design, 50-100% during later phases
- **When needed**: Projects with extensive documentation requirements

### Scrum Master (スクラムマスター)
- **Seniority**: Mid (3+ years Agile experience)
- **Responsibilities**: Facilitate ceremonies, remove impediments, coach team on Agile practices
- **Allocation**: 50-100% (can serve 1-2 teams)
- **When needed**: Agile/Scrum projects
