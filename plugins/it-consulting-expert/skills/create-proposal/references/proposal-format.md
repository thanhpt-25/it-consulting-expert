# Proposal Formatting Guidelines

## Document Standards

### For Japanese Clients
- Use A4 page size
- Font: Meiryo or Yu Gothic for body, size 10-11pt
- Headers: 14pt bold for section titles, 12pt bold for subsections
- Page numbers in footer
- Company logo on cover page
- Formal keigo (敬語) throughout
- Date format: 2024年1月15日 (Japanese) or 2024/01/15
- Currency: ¥ with comma separators (¥1,000,000)

### For International Clients
- Use A4 or Letter size (ask client preference)
- Font: Calibri, Arial, or similar sans-serif, size 11pt
- Professional but not overly formal tone
- Date format: ISO 8601 (2024-01-15) or client-preferred
- Currency: use client's currency with appropriate symbol

## Cover Page Elements
1. Document title (提案書 / Proposal)
2. Client company name (御中)
3. Project name
4. Date
5. Proposing company name, division, and contact
6. Confidentiality notice (秘密保持 / Confidential)
7. Version number

## Section Numbering
Use hierarchical numbering:
- 1. Section
  - 1.1. Subsection
    - 1.1.1. Sub-subsection

## Table of Contents
Include for proposals over 10 pages.

## Page Header/Footer
- Header: project name or document title (left), client name (right)
- Footer: page number (center or right), confidentiality mark (left)

## Diagrams
- Include legends for all diagrams
- Use consistent color coding throughout
- Prefer vector graphics over screenshots
- Label all components clearly

## Version Control
Include revision history table on page 2:
| Version | Date | Author | Changes |
|---------|------|--------|---------|
| 0.1 | YYYY-MM-DD | Name | Initial draft |
| 1.0 | YYYY-MM-DD | Name | Final version |
